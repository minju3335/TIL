> Markdown
- 일반 텍스트 기반의 경량 마크업 언어를 뜻함
- 일반 텍스트로 서식이 있는 문서를 작성하는데 사용
- HTML과 리치 텍스트(RTF) 등 서식문서로 쉽게 변환됨

- 문법
    - `#` : 문단제목
    - `**` : 앞뒤에 적어주면 볼드체
    - `*` : 앞뒤에 적어주면 이탤릭체
    - `[링크이름](링크주소)` : 하이퍼링크
    - `[사진이름](사진인터넷주소)` : 인터넷사진 인용
    - `[사진이름](사진파일주소)` : 컴퓨터사진 인용
    - `1. ` : 번호 부여
    - `>` : 블럭 인용
    - `- ` : 순서없는 리스트
    - ` ``` ``` ` : 다음에 언어를 적으면 해당하는 코드를 적을 수 있음
